# Wedding Site

Свадебный сайт-приглашение Алексея и Анастасии, созданный на `React + TypeScript + Vite`.

## Доступные домены

- `alexeyanastasia2026.ru`
- `alexeyanastasia2026.online`
- `алексейианастасия2026.рф`

## Особенности проекта

- Интро-экран с видео-анимацией, аккуратно встроенной в пользовательский сценарий входа на сайт.
- Невероятно продуманный дизайн: типографика, композиция, адаптивность и плавные анимации работают как единая визуальная система.
- Оптимизированный одностраничный формат для быстрого открытия и комфортного просмотра на мобильных и десктопах.

## Локальный запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
npm run preview
```
